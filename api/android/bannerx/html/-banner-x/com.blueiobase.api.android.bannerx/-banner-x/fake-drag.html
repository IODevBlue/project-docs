@JvmOverloads
fun fakeDrag(offset: Float, duration: Long = 0, delay: Long = 0, interpolator: TimeInterpolator? = null)

Execute a fake drag operation on BannerX.

Parameters

offset

The number of pixels to drag by. A positive offset shows the left Banner and a negative offset shows the right Banner. For example: An offset of 0.5F shows the half of the immediate leftward Banner, an offset of -1 shows the immediate rightward Banner completely, an offset of -2 shows 2 rightward Banner objects etc.

duration

The duration of the fake drag operation.

delay

The amount of delay applied before the fake drag operation starts.

interpolator

Optional animation interpolator to apply to the fake drag operation.
